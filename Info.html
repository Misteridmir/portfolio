HTML e CSS

HTML (HyperText Markup Language) Traduzido para o português Linguagem de Marcação de hipertexto. A Linguagem de marcação como meio de estrutura semântica para a criação de sites, é a principal e mais popular forma de se construir sites para Internet. Conheça sobre a história do HTML em historia sobre os sites de busca

CSS (Cascading Style Sheets) Traduzido para o português Folha de estilo em Cascata, usado como meio de estilizar o Contéudo dos Sites em HTML, segundo a MDN docs é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou em XML (incluindo várias linguagens em XML como SVG, MathML ou XHTML). O CSS descreve como elementos são mostrados na tela, no papel, na fala ou em outras mídias.

JavaScript

Para completar o trio da web, a linguagem favorita do developers Webs que não perdem a oportunidade de usar e abusar da Linguagem. Segundo a Mozilla Concebido como uma linguagem do lado do servidor por Brendan Eich (então empregado pela Netscape Corporation), o JavaScript logo chegou ao Netscape Navigator 2.0 em setembro de 1995. O JavaScript obteve sucesso imediato e o Internet Explorer 3.0 (en-US) introduziu o suporte ao JavaScript sob o nome JScript em agosto de 1996.

A Sua maior Caracteristica é a utilização do DOM (Document Object Model) acesse a pagina da W3School para entender sobre. Mas basicamente dar a linguagem dominio para acessar as CSSs e O elementos do HTML, totalment como objetos, podendo assim modificar a estrutura das paginas e folhas de estilos como o desenvolvendor desejar.
